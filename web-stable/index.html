Підключення до дрона

Кабель (USB) / радіо / мережа. Маршрут — у вкладці «План».

Порожнє = слухати всі мережі на 14550.

Польотник зі SpeedyBee-Bluetooth або BLE-UART модуль. На BT-UART польотника має бути MAVLink (SERIALx_PROTOCOL=2), не MSP.

Якщо політник мовчить по Bluetooth — просто підключись раз по USB-OTG або WiFi: додаток сам налаштує його Bluetooth і перепідключиться.

Маршрут не побудовано.

Керування польотом

Послідовність: ARM → «Старт місії» (AUTO). Тримайте готовність до перехоплення керування.

Пауза тримає дрон на маршруті й на висоті — продовження без ризику.

Кожен виліт логується офлайн і калібрує оцінку часу / заряду.

Статистика польотів
Резервна копія (свій сервер)

Опційно: зберігає поля, журнал польотів і параметри на твоєму сервері, щоб не втратити їх при заміні або втраті телефону. Дані йдуть лише на твій сервер.

вимкнено

Встановити додаток

Офлайн-додаток. Тут завжди актуальна версія — встановив раз, далі працює без інтернету.

Перевіряє сервер на нову версію й оновлює прямо тут: веб-версія — миттєво, APK — завантажить новий і запропонує встановити.

Android — нативний (APK): прямий зв'язок з польотником (STM32 / Pixhawk) по USB-кабелю. Завантаж → дозволь встановлення з невідомих джерел → відкрий.

Android APK

Версія стабільного APK: v2.5.36.

Бета (тестова збірка): нові функції на перевірку. Ставиться ОКРЕМО від стабільної — інша іконка «FMP BETA», стабільну не замінює й не видаляє. Зараз у беті: НОВИЙ польовий інтерфейс — карта на весь екран + великі кнопки по кроках Поле→Маршрут→Дрон→Політ (видно, що тиснути далі); керування ARM/СТАРТ/RTL «протягни-щоб-підтвердити» (захист від випадкових дотиків у полі); фікс пропуску вузьких ділянок поля.

Завантажити Android APK — БЕТА v2.6.6-beta

З версії 2.6.5 бета оновлюється з середини додатку (тапни версію вгорі → «Оновити») — далі не треба качати вручну.

iPhone / iPad (iOS): Apple не дозволяє ставити додатки із сайту, як APK — тому «завантажити й відкрити» на iOS неможливо. Нативний застосунок ставиться лише через: (1) Mac + Xcode по кабелю (безкоштовний Apple ID, сертифікат діє 7 днів) — шлях розробника; або (2) TestFlight (потрібен Apple Developer Program, $99/рік) — тоді тут з'явиться посилання-запрошення. Поки iOS-збірка готова й перевірена, але не опублікована. Зв'язок з дроном на iOS — лише WiFi / ELRS-бекпак (USB Apple забороняє).

iOS — TestFlight (скоро)

ПК (Windows / Mac): просто відкрий цей додаток у Chrome / Edge — повний функціонал: планування + дрон по USB-кабелю (WebSerial). Тисни «Встановити» нижче, щоб був як окремий застосунок і працював офлайн.

У Chrome: меню (⋮) → «Встановити додаток» / «Додати на головний екран».

На Android у браузері — лише ПЛАНУВАННЯ (браузер Android не має доступу до USB). Для дрона на телефоні — APK вище.
