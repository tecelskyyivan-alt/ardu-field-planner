ArduCopter · обприскування поля

1 Задай контур поля

Постав точки контуру: пануй карту під хрестик і тисни ➕ точка, або тапай по карті / бери 📍 свою GPS чи 🛩️ GPS дрона. Тягни вершини, «+» на ребрі додає точку. Натисни ✓ Готово.

Тягни вершини, щоб уточнити межу поля — маршрут перебудовується наживо. Натисни ще раз, щоб завершити.

Завантаж тайли поточного району (або поля), щоб карта працювала без інтернету в полі.

Намалюй полігони-перешкоди над деревами / кущами / дорогами / ставками всередині поля — кожен вирізається з покриття. Клік по вирізу видаляє його.

2 Параметри маршруту

Тягни повзунок — маршрут перебудовується наживо.

Старт і фініш наближаються до обраного якоря — менше транзиту, коротший політ. «GPS дрона» бере позицію/HOME із телеметрії (вкладка «Політ»).

🚁 Підключення до дрона

Кабель (USB) або радіо/мережа. Залий місію напряму й дивись телеметрію. Прошивка ArduPilot. Спершу побудуй маршрут у «Плані».

Залиш порожнім — програма сама слухає всі мережі на порту 14550 (бачить дрон/бекпак/SITL за будь-якою адресою). Заповнюй лише для нестандартного порту.

Маршрут не побудовано.
Керування польотом

Запуск залитої місії: «Старт» переведе в AUTO й почне політ. Будь готовий перехопити апарат. Спершу — увімкнення моторів (ARM).

Кожен реальний виліт (AUTO) записується офлайн. Журнал калібрує оцінку часу/батареї під твій дрон — наступні маршрути рахуються точніше.

⬇️ Встановити додаток

Офлайн-додаток. Тут завжди актуальна версія — встановив раз, далі працює без інтернету.

Перевіряє сервер на нову версію й оновлює прямо тут: веб-версія — миттєво, APK — завантажить новий і запропонує встановити.

📱 Android — нативний (APK): прямий зв'язок з польотником (STM32 / Pixhawk) по USB-кабелю. Завантаж → дозволь встановлення з невідомих джерел → відкрий.

📱 Android APK

Версія стабільного APK: v2.5.4.

🧪 Бета (тестова збірка): нові функції на перевірку. Ставиться ОКРЕМО від стабільної — інша іконка «FMP BETA», стабільну не замінює й не видаляє. Зараз у беті: малювання контуру та вирізу в агро-стилі (хрестик у центрі + кнопка «точка», додавання з GPS, перетягувані вершини, «+» на ребрі).

🧪 Android APK — БЕТА v2.6.1-beta

💻 ПК (Windows / Mac): просто відкрий цей додаток у Chrome / Edge — повний функціонал: планування + дрон по USB-кабелю (WebSerial). Тисни «Встановити» нижче, щоб був як окремий застосунок і працював офлайн.

У Chrome: меню (⋮) → «Встановити додаток» / «Додати на головний екран».

🌐 На Android у браузері — лише ПЛАНУВАННЯ (браузер Android не має доступу до USB). Для дрона на телефоні — APK вище.
